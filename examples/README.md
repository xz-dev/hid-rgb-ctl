# Examples

Scripts that use `hid-rgb-ctl` for keyboard lighting effects.

## Included

- **rainbow.sh** — Smooth rainbow gradient loop cycling through all hues.

## Contributing

Got a cool lighting effect? PRs welcome! Just drop a script here.

Some ideas:
- Breathing / pulse effect
- Color temperature shift (warm ↔ cool)
- Reactive to CPU/GPU temperature
- Music visualizer (pipe audio levels in)
- Pomodoro timer (color changes on work/break)
- Battery level indicator

Scripts can be in any language — just make sure they call `hid-rgb-ctl set`.
